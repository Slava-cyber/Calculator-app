The documentaion of SMARTCALC_V1.0

SMARTCALC_V1.0 - is implementation of extended version of the usual calculator. Calculating arbitrary bracketed arithmetic expressions in infix notation. Plotting a graph of a function given by an expression in infix notation with the variable x.

Functional

Arithmetic opeartors

  • Brackets (a + b)
  • Additional a + b
  • Subtraction a - b
  • Multiplication a * b
  • Division a / b
  • Power a ^ b
  • Modulus a % b
  • Unary plus +a
  • Unary minis -a

Functions

  • cos(x) cosine
  • sin(x) sine
  • tan(x) tangent
  • acos(x) arc cosine
  • asin(x) arc sine
  • atan(x) arc tangent
  • sqrt(x) square root
  • ln(x) natural logarithm
  • log(x) common logarithm

how to use

  1. To calculate an expression without x, you need to click on the button"="
  2. To evaluate an expression with x, you need to enter an expression with x, then press "=", then enter the value of x and press again "="
  3. To draw a graph, you need to click on the button f(x)
  4. Error always "ERROR"

Recomendation

  1. Do not enter invalid values. Example: cos()
